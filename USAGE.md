## Usage

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run dev server:
   ```bash
   npm run dev
   ```

3. Environment variables:
   - NEXT_PUBLIC_API_URL
   - NEXT_PUBLIC_SENTRY_DSN

Theme:
- Uses `next-themes` with `class` attribute. The ThemeToggle component is provided.

Notes:
- Add your shadcn/ui components into `src/components/ui/` if you want.
