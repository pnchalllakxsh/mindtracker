# Next.js Boilerplate

Generated starter for Next.js (App Router) + TypeScript + Tailwind + shadcn-like UI + Radix + React Query + Zustand + React Hook Form + Zod + Axios + Sentry

Features:
- App Router (`app/` folder)
- Tailwind with Light/Dark + system support and a manual toggle
- Axios instance with interceptors
- React Query client provider
- Zustand example store
- Basic Sentry init file (configure DSN via env)
